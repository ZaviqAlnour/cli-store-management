# Store Management CLI

A simple command-line store management program written in C.

## Features
- Menu-based command-line interface
- Add new products
- View all products
- Update product price
- Remove a product

## Status
This is an early and simple version of the project.  
More features may be added in the future, or it may remain a basic CLI tool for learning purposes.

## How to Run
1. Compile the program using a C compiler:
